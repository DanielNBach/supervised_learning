SETUP:
The data should be placed in a folder "data" relative to the notebook and named breast_cancer_data.csv and iris_.csv.
Data can be downloaded at:
https://www.kaggle.com/datasets/uciml/breast-cancer-wisconsin-data
https://www.kaggle.com/datasets/uciml/iris
scikit-learn should be on version 1.2.2.
To run the code, simply open the notebook using Jupiter Notebook or Google Colab and run all cells. Figures will be saved.
